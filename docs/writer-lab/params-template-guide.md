# 自定义参数模板使用指南

> 写作风格 Skill 能让 AI 学会"怎么写"，但"写给谁"、"用什么叙事手法"、"要不要用网络用语"——这些创作决策只有你自己知道。自定义参数模板就是用来填补这个空白的。

---

## 为什么需要自定义参数模板？

智子X 的[写作建模体系](modeling-dimensions-reference.md)有 14 个维度，但它们被分成了两类：

**AI 自动提取的 6 个维度**（生成 Skill 时自动完成）：
- 表层风格（词汇、句法、段落、篇章、语用）
- 修辞手法（比喻、排比、反问密度等）
- 情感曲线（情绪起伏轨迹）
- 内容结构（总分总、递进式等）
- 认知模式（归纳/演绎/类比）
- 节奏韵律（句子节奏、流畅度）

这 6 个维度是"风格特征"——可以从样本文章中稳定提取，跨场景复用。比如"鲁迅风格"可以写任何主题。

**用户指定的 8 个维度**（就是自定义参数模板）：
- 叙事技巧、论证策略、文化语境、读者定位、特殊要求、读者互动、多模态特征、语言变异

这 8 个维度是"创作决策"——每次写作的需求都不同，AI 无法从样本中推断。比如同一个"鲁迅风格"，写给专家和写给小学生，参数完全不同。

**一句话总结：Skill 决定"像谁写的"，模板决定"怎么写给谁"。**

---

## 什么时候用？什么时候不用？

**建议使用**：
- 你对文章有明确的风格要求（比如"要口语化、用emoji、写给年轻人"）
- 你反复写同一类内容（比如每周写公众号），想固定一套参数
- 你觉得 AI 生成的文章"方向不对"，想精确控制

**可以不用**：
- 随便写写，没有特别要求 → 选"不使用模板"即可
- 已经选了 Skill + 参考文章，效果已经满意

## 8 个维度怎么填？

不需要全部填写。根据你要写的内容类型，挑相关的维度填就行。

### 1. 叙事技巧（写故事/散文时用）

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 叙事视角 | 谁在讲故事 | 个人经历选"第一人称"，写他人故事选"第三人称" |
| 时间结构 | 故事怎么展开 | 正常讲选"顺叙"，先说结局再回溯选"倒叙" |
| 对话比例 | 对话占多少 | 0.1=几乎没对话，0.3=一般小说，0.7=剧本感 |
| 场景比例 | 描写多细 | "高"=慢节奏细腻描写，"低"=快速推进 |

### 2. 论证策略（写评论/议论文时用）

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 推理方式 | 怎么论证 | 从理论推结论选"演绎"，从案例总结规律选"归纳" |
| 证据类型 | 用什么支撑 | 数据说服力强，案例更生动，权威引用增加可信度 |
| 反驳风格 | 怎么处理反对意见 | "先让步再反驳"最稳妥，"直接反驳"更有力 |

### 3. 文化语境

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 文化引用 | 引用什么文化 | 填你想要的，比如"中国传统文化"、"流行文化、网络梗" |
| 时代背景 | 什么年代感 | 大多数选"现代"，写古风选"古代" |
| 地域特色 | 有没有方言味 | "东北"=豪爽直率，"江南"=温婉细腻，不需要就选"其他" |

### 4. 读者定位

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 专业水平 | 读者懂多少 | "专家"=可以用术语，"初学者"=从零解释 |
| 解释程度 | 解释多详细 | 配合专业水平选，给初学者就选"详细解释" |
| 假设已知概念 | 哪些不用解释 | 填读者肯定懂的词，比如"互联网、浏览器" |

### 5. 特殊要求

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 约束条件 | 必须遵守的规则 | 比如"每段不超过3行"、"必须包含3个案例" |
| 强调重点 | 要突出什么 | 比如"强调实用性"、"增加幽默感"、"突出创新性" |

### 6. 读者互动

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 直接称呼读者 | 用不用"你" | 公众号文章建议开，学术论文建议关 |
| 提问频率 | 多久问一次 | 0.1=很少提问，0.3=适度，0.7=频繁互动 |
| 互动风格 | 什么语气 | "对话式"=朋友聊天，"引导式"=老师教学 |
| 共鸣点 | 触动什么情感 | 比如"打工人的心酸"、"父母的不易" |

### 7. 多模态特征

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 标点风格 | 标点怎么用 | "规范"=正式，"随意"=多省略号，"强调式"=多感叹号 |
| 表情符号 | 用不用 emoji | 自媒体开，正式文章关 |
| 网络用语 | 用不用网络梗 | "yyds"、"绝绝子"这类，看受众决定 |
| 段落间距 | 段落多密 | 手机阅读选"宽松"，长文选"紧凑" |

### 8. 语言变异

| 参数 | 说明 | 怎么选 |
|------|------|--------|
| 正式度变化 | 正式和口语混着来 | 开了之后文章会更有层次感 |
| 语域切换 | 切换语言风格 | 比如"学术语域、口语语域"混用 |
| 多语言混用 | 夹杂外语 | 比如"中英混用"，适合科技类文章 |

## 内置模板速查

不想自己配？直接用内置模板：

| 模板 | 适合写什么 | 核心特点 |
|------|-----------|---------|
| 📖 故事叙事型 | 小说、散文、故事 | 第一人称、对话30%、细腻描写、宽松排版 |
| 💼 专业评论型 | 评论、议论文、深度分析 | 演绎推理、数据论证、专家受众、规范标点 |
| 🎉 轻松自媒体型 | 公众号、短视频文案 | 口语化、emoji、网络梗、高互动、幽默 |
| 🎓 学术严谨型 | 学术论文、技术文档 | 第三人称、权威引用、最少解释、紧凑排版 |

你也可以基于内置模板修改，保存为自己的模板。
