# 为什么智子X的媒体解析需要安装扩展？

> 适用读者：所有使用媒体解析工具的用户

---

## 先说结论

**安装扩展后，你可以获得：**

- ✅ **无限次数解析** — 不受服务器配额限制
- ✅ **高速解析** — 你的浏览器直接从平台获取资源，不经过中转
- ✅ **隐私保护** — 我们的服务器不储存你的任何信息
- ✅ **稳定可靠** — 解析的稳定性只取决于你的网络
- ✅ **大部分平台无需登录** — B站、快手、抖音完全无感解析，只有小红书需要登录

---

## 一、其他解析网站怎么做的？

你可能用过的那些视频解析网站——粘贴链接就能下载，不需要扩展程序或者登录认证。

**他们的原理：**
- 他们准备了几十上百个平台账号和IP地址池进行轮询
- 你提交链接后，网站用这些账号去平台下载
- 下载完成后，通过网站服务器转发给你

**这种方案的问题：**

```text
| 问题 | 说明 |
|------|------|
| 隐私泄露 | 你的每个解析请求都经过网站服务器，网站知道你解析了什么 |
| 次数限制 | 共享账号容易被平台检测，网站必须限制每人使用次数 |
| 解析慢 | 需要经过服务器中转，速度取决于服务器带宽 |
| 成本高 | 需要维护账号池、IP池、服务器，每月成本几千元 |
```

**那些"免费"网站怎么回本？**
- 投放广告（强制用户播放点击广告）
- 收集用户数据转卖
- 引流到付费服务
- 其他商业目的

## 二、智子X 怎么做的？

智子X 采用**哑管道架构**（Dumb Pipe），根据平台特性使用不同策略：

**三种解析策略：**

1. **静默解析**（B站、快手）
   - 扩展直接抓取平台网页，解析内嵌数据
   - 完全无感，不需要任何用户操作
   - 不需要登录，不需要弹窗

2. **智能注入**（抖音）
   - 扩展在平台页面执行解析脚本
   - 自动处理平台的反爬签名
   - 不需要用户登录，但会短暂打开平台页面

3. **授权解析**（小红书）
   - 需要你在小红书官网登录一次
   - 登录后，平台给你的浏览器发一个临时通行证（Cookie）
   - 扩展读取这个通行证，存在你电脑本地
   - 解析时用这个通行证直接向平台请求数据

**关键点：**
- **快速，稳定，只取决于你的网络环境**
- **服务器不储存你的任何信息** — Cookie、解析记录、账号信息都在你本地
- 解析请求从你的浏览器直接发给平台，不经过中间人

## 三、平台登录需求说明

```text
| 平台 | 是否需要登录 | 解析方式 | 说明 |
|------|------------|---------|------|
| B站 | ❌ 不需要 | 静默解析 | 扩展自动抓取网页数据，完全无感 |
| 快手 | ❌ 不需要 | 静默解析 | 扩展自动抓取网页数据，完全无感 |
| 抖音 | ❌ 不需要 | 智能注入 | 扩展自动处理，会短暂打开平台页面 |
| 小红书 | ✅ 需要 | 授权解析 | 需要在小红书官网登录一次 |
```

**重点** 即使小红书需要登录，Cookie 也只存在你的电脑上，不上传到任何服务器。

## 四、两种方案对比

```text
| 特性 |          服务器API方案             | 扩展方案（哑管道）    |
|------|-----------------------------------|---------------------|
| 谁的账号 | 解析平台的共享账号               | 有些资源需要自己账号 |
| 请求路径 | 你 → 服务器 →平台 → 服务器 → 你  | 你 → 平台 → 你      |
| 隐私     | 网站知道你的所有解析记录         | 没有人知道          |
| 次数限制 | 有限制（共享账号易被检测）        |    无限制           |
| 解析速度 | 慢（经过服务器中转）             |   快（直接获取）     |
| 服务稳定性 | 低（共享账号被封全员受影响）     |   高（独立环境）    |
| 是否需要登录 |          不需要              |     部分资源需要    |
```

## 五、为什么 智子X 不提供服务器API？

**我们提供了，但只作为降级方案，且未来可能关闭。**

### 成本问题

服务器API方案需要：
- 准备几十上百个平台账号，轮流使用避免被封
- 购买代理IP池，避免单一IP被封禁
- 服务器带宽、存储、计算资源
- 持续维护：账号被封要补充，IP被封要更换

**这些成本加起来，每月至少几千元。**

### 智子X 的定位

智子X 是 **Open Core 项目**（核心开源），我们的原则是：

- 最大限度保护用户隐私
- 不依赖高成本的服务器方案
- 提供可持续的免费服务

### 扩展方案更可持续

- 你用自己的账号，我们不需要维护账号池
- 你用自己的网络，我们不需要购买IP池
- 请求从你本地发出，我们不需要服务器带宽

**这样，我们可以把精力放在开发更好的功能上。**

## 六、总结

**原来的服务器API方案：**
- 优点：不需要登录，方便
- 缺点：有次数限制、解析慢、隐私风险、服务器IP易被封、成本高

**扩展方案：**
- 优点：无限次数、高速（只取决于你的网络）、隐私保护、服务稳定
- 缺点：部分资源需要账号登录状态才能获取

**智子X 的选择：**
- 作为 Open Core 项目，我们选择更可持续的扩展方案
- 不收集用户数据，不强插广告
- 服务器API只作为降级方案，未来可能关闭
- 多一步安装扩展，换来的是无限次数、高速解析、隐私保护
